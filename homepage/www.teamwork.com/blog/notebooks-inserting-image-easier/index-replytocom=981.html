Notebooks – Inserting an image has never been easier.

Posted May 23, 2014

It feels really good to cross this oneÂ off our to-do list! One of the top feedback requests that weâ€™ve been getting for some time now has been the option to add an image to a Notebook without having to embed it using the URL of an image on another server.

This month we went ahead and did it.Â You can now upload an image from your computer and inset it in to your notebook.

So, how do you do this?

1) Create the Notebook in the same way as before.

2) Select the option from the editor bar to insert or edit an image.

3) Select the green arrow if you want to upload from your local machine.

4) Select the file and choose to upload the image.

Any image that works on the web will be accepted (jpg, png or gif). Because these are being inserted in to notebooks we’ll resize the image if it’s bigger than 1024 pixels in width.
